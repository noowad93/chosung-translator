# Python 3.7 Template

스캐터랩에서 사용하는 Python 3.7 레포지토리 템플릿입니다!

## 포함되어 있는 사항

- black, isort, flake8 세팅
- pyright 세팅
- pytest, pytest-cov 세팅
- CircleCI 세팅
- tox 세팅
- 권장 vscode extension 목록 제공
- 이슈 템플릿
- PR 템플릿
- README.md
- 우리의 사랑 💛

## 레포지토리 생성 후 바꾸어야 할 사항

- 새로 작성하기
  - `README.md`
  - `pack/`
  - `tests/`
- 이름 바꾸기
  - `pyrightconfig.json`
  - `.circleci/config.yml`
  - `setup.py`
  - `tox.ini`

## pyright

혹시라도 바꾸어 사용하시고 싶으신 분들은 [pyright - configuration.md](https://github.com/microsoft/pyright/blob/master/docs/configuration.md)에서 자세한 설정법을 볼 수 있습니다.
